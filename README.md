# Projenin İçeriği
Kurumsal mimari yapısı kullanılarak *Clean Code* ile *SOLID* kuralları ile oluşturulmuştur.<br/> Angular ile yazılmış, Araba kiralama fikri üzerine oluşmuştur.
- CRUD Operasyonlar
  - Add
  - Read
  - Update
  - Delete
- Filtreleme
- Auth
  - Login
  - Register
  - Profile Update
  - Password Update
- Reactive Forms
- Validasyon işlemleri
  - Toastr
- Models
- Services
- Custom pipes
